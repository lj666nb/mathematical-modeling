### 1.2 双模式复用

同一个 Agent 在学习模式和竞赛模式下的行为不同：

| Agent | 学习模式行为 | 竞赛模式行为 |
|-------|-----------|------------|
| 数学答疑 | 概念解释+举例说明+追问引导 | 快速判断+精简回答+指向引擎 |
| 建模引导 | 逐步引导+教选择逻辑+分析优缺点 | 确认选题+注意事项+交给引擎 |
| 代码辅助 | 纠错+解释+教怎么写 | 快速修复+性能提示 |
| 论文评审 | 结构化评语+改进方向+范文参考 | 门禁报告解读+修改建议 |

## 二、Agent 基类设计（已有，改造点说明）

现有 `agent_base.py` 已提供：

```python
class BaseAgent(ABC):
    def __init__(self, db: AsyncSession, user_id: int)
    async def get_llm_config(config_id) -> dict    # 读取用户LLM配置
    async def call_llm(messages, system_prompt)     # 统一LLM调用
    def build_context_messages(message, history)     # 构建上下文
    @abstractmethod
    async def process(message, **kwargs) -> dict    # 子类实现
```

**改造点**：
1. `call_llm()` 增加 `user_profile` 参数，根据用户画像调整 temperature/回答风格
2. `build_context_messages()` 增加 token 预算管理（保留最近 N 轮，超过则压缩）
3. `process()` 返回值增加 `suggestions` 字段（追问建议）

## 三、Dispatcher 改造：增加意图识别

```python
class AgentDispatcher:
    # Agent 类型映射（改造）
    AGENT_MAP = {
        "math-qa": MathQAAgent,
        "modeling-guide": ModelingGuideAgent,
        "code-assist": CodeAssistAgent,
        "paper-review": PaperReviewAgent,
    }

    # 🆕 意图识别规则（自动路由）
    INTENT_RULES = [
        (r"(代码|报错|bug|error|怎么写|编程|python|matlab)", "code-assist"),
        (r"(论文|摘要|评审|审稿|格式|结构)", "paper-review"),
        (r"(选什么模型|用什么方法|这道题|建模思路|假设|分析)", "modeling-guide"),
        # 兜底 → math-qa
    ]

    async def auto_detect_agent(self, message: str) -> str:
        """根据用户消息自动推荐合适的 Agent"""
        for pattern, agent_type in self.INTENT_RULES:
            if re.search(pattern, message, re.IGNORECASE):
                return agent_type
        return "math-qa"

    async def dispatch(self, agent_type, message, session_id, **kwargs):
        # 1. 加载上下文 (Redis → fallback DB)
        context = await self._load_context(session_id)

        # 2. 🆕 加载用户画像
        user_profile = await self._load_user_profile()

        # 3. 获取 Agent 实例
        agent = self._get_agent(agent_type)

        # 4. 调用处理（传入画像）
        result = await agent.process(
            message=message, session_id=session_id,
            history=context, user_profile=user_profile, **kwargs
        )

        # 5. 保存上下文
        await self._save_context(session_id, context)

        return result
```

## 五、Agent 知识加载机制

```
                     ┌─────────────────────┐
                     │  Agent.process()     │
                     │  调用 build_prompt() │
                     └──────────┬──────────┘
                                │
              ┌─────────────────┼─────────────────┐
              ▼                 ▼                   ▼
    ┌─────────────┐   ┌──────────────┐   ┌─────────────────┐
    │ System      │   │ Domain       │   │ User Profile    │
    │ Prompt      │   │ Knowledge    │   │ Context         │
    │ (Agent专属)  │   │ (公共知识库)  │   │ (画像适配)       │
    └──────┬──────┘   └──────┬───────┘   └───────┬─────────┘
           │                 │                    │
           ▼                 ▼                    ▼
    ┌────────────────────────────────────────────────────┐
    │          build_prompt() 拼接                      │
    │  f"""                                                │
    │  {SYSTEM_PROMPT}                                    │
    │                                                     │
    │  ## 知识参考                                         │
    │  {RELEVANT_KNOWLEDGE}  ← 从 domain_knowledge 检索    │
    │                                                     │
    │  ## 用户画像                                         │
    │  {USER_PROFILE_ADAPTATION} ← 从 user_profiles 匹配   │
    │                                                     │
    │  ## 教学准则（所有Agent共享）                         │
    │  {TEACHING_PRINCIPLES} ← 从 teaching_guidelines 加载 │
    │  """                                                │
    └────────────────────────────────────────────────────┘
```

### 知识检索策略（简易 RAG）

```python
def retrieve_relevant_knowledge(agent_type: str, message: str) -> str:
    """
    根据 Agent 类型和用户消息，检索相关知识片段：
    - math-qa: 关键词匹配 MATH_TERMINOLOGY + MODEL_CLASSIFICATIONS
    - modeling-guide: 关键词匹配 MODEL_SELECTION_DECISION_TREE + MODEL_CLASSIFICATIONS
    - code-assist: 关键词匹配模型→对应代码模板
    - paper-review: 加载 COMPETITION_INFO 中的评分标准
    """
```
