| Markdown 渲染 | **markdown-it** + **KaTeX** | 数学公式支持 |
| 复杂交互组件 | **React 18**（按需嵌入） | 建模画布、交互式图表 |

### 为什么 Vue3 为主 + React 按需嵌入？

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| 纯 Vue3 | 统一技术栈，包体积小 | 建模画布、复杂可视化生态不如 React | 适用于 80% 页面 |
| 微前端(qiankun) | 完全隔离，独立部署 | 太重，MVP 阶段不值得 | ❌ |
| **Vue3 为主 + iframe/Web Component 嵌入 React** | 平衡统一性和灵活性 | React 组件需单独打包 | ✅ 选用 |
| 纯 React | React 生态最丰富 | 需重写已有 Vue 代码 | ❌ |

**具体嵌入方式**：React 组件独立打包为 UMD/Custom Element，在 Vue 页面中通过 `<react-wrapper>` 组件加载。

## 三、Vue ↔ React 集成方案

```
┌────────────────────────────────────────────┐
│              Vue3 页面                      │
│                                            │
│  <template>                                │
│    <react-wrapper                          │
│      component="InteractiveChart"          │
│      :props="{ data, options }"            │
│      @chart-click="handleClick"           │
│    />                                      │
│  </template>                               │
│                                            │
│  ┌──────────────────────────────────────┐  │
│  │  <react-wrapper> (Vue3 包装组件)      │  │
│  │                                      │  │
│  │  mounted() →                          │  │
│  │    ReactDOM.createRoot(el)            │  │
│  │      .render(<ChartComponent />)      │  │
│  │                                      │  │
│  │  updated() → props diff → re-render  │  │
│  │  unmounted() → root.unmount()        │  │
│  └──────────────────────────────────────┘  │
└────────────────────────────────────────────┘
```

**React 组件加载流程**：
1. `react-components/` 独立打包为 `react-widgets.js`（UMD）
2. `index.html` 通过 `<script>` 预加载（或动态 import）
3. 每个 React 组件注册为 Custom Element：`<interactive-chart>`, `<modeling-canvas>`
4. Vue3 中通过 `<react-wrapper>` 通用包装组件使用

**备选方案**（如果 Custom Element 方案复杂）：iframe 嵌入 + postMessage 通信——简单但性能略差，适用于 MVP。

## 四、构建配置

### vite.config.js 关键配置

```js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [
    vue(),
    // React 仅用于 src/react-components/ 目录
    react({ include: 'src/react-components/**' }),
  ],
  server: {
    port: 5173,
    proxy: {
      '/api': 'http://localhost:8000',
      '/ws': { target: 'ws://localhost:8000', ws: true }
    }
  },
  build: {
    rollupOptions: {
      input: {
        main: 'index.html',
        // React 组件独立入口（可选）
        'react-widgets': 'src/react-components/index.js'
      }
    }
  }
})
```
